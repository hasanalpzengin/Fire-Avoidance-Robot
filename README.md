# Fire Avoidance Robot
Computer Vision @ - https://github.com/digitalbrain79/darknet-nnpack/
Yolov3 Lite Dataset @ - https://pjreddie.com/darknet/yolo/
Paho's Python MQTT Library @ - https://pypi.org/project/paho-mqtt/

##### Components
- Raspberry Pi 3
- Mq-2
- TCRT5000
- Raspberry Camera
- DC Motor
- L298N Motor Driver
